Text Button directive documentation

Uk2TextButtonDirective is a directive that will only work on <button> elements with mat-button directive, and it will add a class to handle styles.

Requisites

You need to import the Uk2DirectivesModule from @axos/uikit-v2-lib into the desired module of your Angular App.

    
        import Uk2DirectivesModule from @axos/uikit-v2-lib
