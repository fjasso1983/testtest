Checkbox directive documentation

Uk2CheckboxDirective is a directive that will only work on <mat-checkbox> elements and it will add a class name to handle styles.

Default checkbox

      
        <mat-checkbox  uk2Checkbox  disableRipple > </mat-checkbox>

Skeleton checkbox


      
        <mat-checkbox  uk2Checkbox disableRipple  [uk2IsLoading]="isLoading"> </mat-checkbox>

Indeterminate checkbox

      
        <mat-checkbox  uk2Checkbox disableRipple [indeterminate]="true" > </mat-checkbox>

Disabled checkbox

      
        <mat-checkbox  uk2Checkbox disableRipple [disabled]="true" > </mat-checkbox>

Label

You should not handle the text with the native functionality of the Angular material component. We are going to use a separate <label> component 

      
        <div class="uk2-checkbox-container" >
          <mat-checkbox  uk2Checkbox disableRipple > </mat-checkbox>
          <label > Example label </label>
        </div>
