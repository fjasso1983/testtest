UiKit V2 Datepicker documentation

Uk2DatePicker is a directive that will control the display behavior of <mat-datepicker> element

Requisites

  • You need to import the SASS file in your Angular App styles.scss . 
  
    @import '~@axos/uikit-v2-lib/src/assets/scss/uk2-theming.scss';
  • Directive must be wrapped inside a mat-form-field element or it will throw an error, see example code below
