Modal component documentation

UkModalComponent is a component that can be used to open modal dialogs with Axos styles and animations.

Requisites

You need to import the SASS file in your Angular App styles.scss and include theming-uk2-input with your custom theme. 

  
    @import '~@axos/uikit-v2-lib/src/assets/scss/uk2-theming.scss';
    @include theming-uk2-input($uikit-v2-docs-theme);
