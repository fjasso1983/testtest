Tile state actions

Fixed height:

To achieve gradient scroll in tile content change <mat-card> css property height to a fixed value (e.g. 300px)

Handle click events in header title and chevron icon:

uk2-tile-header-link component has an output to handle chevron clicks and additionally client can add click events in the projected elements inside title section

Projecting header elements:

uk2-tile-header-link component accept content projection in three parts

  1. Title section
  2. Sub-line section
  3. Tooltip content section

Be aware that some elements that will be projected have spacing styles and that can break the alignment with the icons, remember to remove spacing styles on the first child and last child of projected elements
