Value movement documentation

Uk2ValueMovementComponent is created based on Angular Material components library, enhanced with custom behavior and styling. It can be used specifying thenext input parameters.

Requisites

  
    @import Uk2ValueMovementType.
    @import Uk2IconValueMovement (There would be a change in the way the icons are organized in the Library).
    @import Uk2IconRegistryService from '@axos/uikit-v2-lib' to registry in the app.component the icons that will be used 
    Example: iconRegistryService.registerIconsFromEnumValues(Object.values(Uk2IconValueMovement), '../assets/svg/icons');
    @import ValueMovementComponent into the component module in which be located the component that will implement the uk2-value-movement control. 
    Note: Until the change takes effect, the icons will have to be added to the project in which the control will be used.

This control accept four parameters:

@ Amount.

It is a string value. Represent the dolars amount formated a USD currency.

@ Percentage.

It is a string value. It represents a percentage, attaching the % character.

@ Description.

It is a string value. It is the text describing the amount and percentage.

@ ValueMovementType.

It is an enum. This enum will let to change the control state.

@ isLoading.

It is a boolean value. It indicates an undefined state, useful when waiting for a service response or from a process to finish.

This is an live example on how it behaves when switching from true to false.

{{ isLoading ? 'Skeleton display' : 'Regular display' }}

@ isEmpty.

It is a boolean value. It indicates is an empty state.

This is an live example on how it behaves when switching from true to false.

{{ isEmpty ? 'Skeleton display' : 'Regular display' }}

@ isIconOnly.

It is a boolean value. It indicates is an icon only state.

This is an live example on how it behaves when switching from true to false.

{{ isIconOnly ? 'Skeleton display' : 'Regular display' }}
