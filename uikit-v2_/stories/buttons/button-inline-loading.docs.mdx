## Inline loading state inputs

Inline loading is a different way to enable loading, that disables the button and shows a custom string with three dots animation, to enable this loading state add the next attributes to button element.

<ul>
  <li>[uk2InlineLoadingStateText] <u>string</u> Custom text that will replace the original button's text when inline loading state is active</li>
  <li>[uk2IsInlineLoadingActive] <u>boolean</u> Toggle to switch from default state to inline loading state.</li>
</ul>
