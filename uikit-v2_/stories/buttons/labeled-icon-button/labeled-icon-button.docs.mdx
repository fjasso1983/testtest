# Uk2LabeledIconButton directive documentation

## Introduction
<p>
<strong>Uk2LabeledIconButton</strong> is a directive designed to customize the styling of <strong>mat-button</strong> directive from <a target="_blank" href='https://v15.material.angular.io/components/button/overview'>Angular Material</a> to adhere to <a target="_blank" href="https://www.figma.com/file/xNJe50LNYYGhQ1Q1C3zs4b/%F0%9F%91%B7%E2%80%8D%E2%99%82%EF%B8%8F-Framework-Core---Web-2.0?type=design&node-id=3921-48808&t=AuhVNSMyDdctrpKb-0">Axos Design System</a>.
</p>

### Constraints
<ol>
  <li>
    It should only be applied to <strong>&lt;button&gt;</strong> elements with  mat-button directive from <a target="_blank" href='https://v15.material.angular.io/components/button/overview'>Angular Material</a>. It will add the <strong> uk2-labeled-icon-button-[large,medium,small] </strong> css class name to handle styles.
  </li>
  <li>
    It should only be applied to <strong>&lt;button&gt;</strong> elements with disableRipple directive. To avoid breaking the predefined styles.
  </li>
  <li>
    White label styling is only supported <strong>without</strong> setting the "color" attribute for the <a target="_blank" href='https://v15.material.angular.io/components/button/overview'>Angular Material</a> component.
  </li>
  <li>
    A label text is required for Labeled Icon Button.
  </li>
  <li>
   Only Icons from Tier1 are supported for Labeled Icon Button.
  </li>
</ol>

<PrerequisitesDoc></PrerequisitesDoc>

<p>
  Note that this directive uses icons and these are part of the @axos/uikit-v2-lib iconography. To see more information go to <a href="/docs/iconography--iconography">iconography.</a>
</p>

## Inputs
<ul>
  <li><strong>[uk2ButtonSize]</strong> <u>Uk2ButtonSizeEnum</u> <i>(optional, <strong>default: Uk2ButtonSizeEnum.small</strong>)</i> is the size of the button. It could be medium or small (Note: large value is not supported.).</li>
  <li><strong>[uk2IsLoading]</strong> <u>boolean</u> <i><i>(optional, <strong>default: false</strong>)</i></i> Toggle to switch from default state to skeleton state.</li>
  <li><strong>[disabled]</strong> <u>boolean</u> <i>(optional, <strong>default: false</strong>)</i> Toggle to disabled state when this is true.</li>
  <li><strong>[uk2LabeledIconPosition]</strong> <u>Uk2LabeledIconPositionEnum</u> <i>(optional, <strong>default: Uk2LabeledIconPositionEnum.right</strong>)</i> Sets the icon position, relative to the label.</li>
</ul>

## Component-specific CSS variables

<ul>
  <li><strong>[--uk2-labeled-icon-button-color]</strong> <i>(default: --uk2-neutral-slate-400)</i> is the labeled icon button text color.</li>
</ul>


## Enums

-  You need to import Uk2ButtonSizeEnum from @axos/uikit-v2-lib into the desired component of your Angular App. To be able to handle the size of button.

<Source
  dark
  language='ts'
  code={dedent`
      import { Uk2ButtonSizeEnum } from '@axos/uikit-v2-lib';
      \r
      export enum Uk2ButtonSizeEnum {
        small = 'small',
        medium = 'medium',
        large = 'large',  // Not supported for Labeled Icon Button.
      }
  `}
/>


-  You need to import Uk2LabeledIconPositionEnum from @axos/uikit-v2-lib into the desired component of your Angular App. To be able to handle the position of the icon.
<Source
  dark
  language='ts'
  code={dedent`
      export enum Uk2LabeledIconPositionEnum {
        left = 'left',
        right = 'right',
      }
  `}
/>

## Sample Code
### .component.html
<p>
  Note the use of <strong>uk2LabeledIconButton</strong> directive, applied to the <strong>&lt;button&gt;</strong> element.
</p>

<Source
  dark
  code={dedent`
    <button
      uk2LabeledIconButton
      mat-button
      disableRipple
      [uk2ButtonSize]="uk2ButtonSize"
      [uk2IsLoading]="uk2IsLoading"
      [disabled]="isDisabled"
      [uk2LabeledIconPosition]="labeledIconPosition"
    >
      {{ label }}
      <mat-icon [svgIcon]="svgIconName"></mat-icon>
    </button>
  `}
/>

### .component.ts
<p>
  Note that svgIconName is part of the @axos/uikit-v2-lib iconography. To see more information go to <a href="/docs/iconography--iconography">iconography.</a>
</p>

<Source
  dark
  language='typescript'
  code={dedent`
    import { Uk2ButtonSizeEnum, Uk2Tier1FilesEnum, Uk2Tier3Enum, Uk2LabeledIconPositionEnum } from '@axos/uikit-v2-lib';

      uk2IsLoading = false;
      isDisabled = false;
      uk2LabeledIconPosition = Uk2LabeledIconPositionEnum.right;
      svgIconName: any = Uk2Tier1FilesEnum.pencil;
      label = 'edit';
      \r
      constructor() {}
    }
  `}
