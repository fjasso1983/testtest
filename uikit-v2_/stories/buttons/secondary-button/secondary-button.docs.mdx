# uk2SecondaryButton directive documentation

## Introduction
<p>
<strong>Uk2SecondaryButton</strong> is a directive designed to customize the styling of <strong>mat-stroked-button</strong> directive from <a href='https://v15.material.angular.io/components/button/overview'>Angular Material</a> to adhere to <a href="https://www.figma.com/file/xNJe50LNYYGhQ1Q1C3zs4b/%F0%9F%91%B7%E2%80%8D%E2%99%82%EF%B8%8F-Framework-Core---Web-2.0?node-id=752%3A27363&t=zlTHs2LZclf3UA1Z-0">Axos Design System</a>.
</p>

### Constraints
<ol>
  <li>
    It should only be applied to <strong>&lt;button&gt;</strong> elements with  mat-stroked-button directive from Angular Material. It will add the <strong> uk2-secondary-button-[large,medium,small] </strong> css class name to handle styles.
  </li>
  <li>
    It should only be applied to <strong>&lt;button&gt;</strong> elements with disableRipple directive. To avoid breaking the predefined styles.
  </li>
  <li>
    White label styling is only supported <strong>without</strong> setting the "color" attribute for the Angular Material component.
  </li>
</ol>
