# Uk2TextIconButton directive documentation

## Introduction
<p>
<strong>Uk2TextIconButtonDirective</strong> is a directive designed to customize the styling of <strong>mat-button</strong> directive from <a target="_blank" href='https://v15.material.angular.io/components/button/overview'>Angular Material</a> to adhere to <a target="_blank" href="https://www.figma.com/file/xNJe50LNYYGhQ1Q1C3zs4b/%F0%9F%91%B7%E2%80%8D%E2%99%82%EF%B8%8F-Framework-Core---Web-2.0?node-id=752%3A27363&t=zlTHs2LZclf3UA1Z-0">Axos Design System</a>.
</p>

### Constraints
<ol>
  <li>
    It should only be applied to <strong>&lt;button&gt;</strong> elements with  mat-button directive from <a target="_blank" href='https://v15.material.angular.io/components/button/overview'>Angular Material</a>. It will add the <strong> uk2-text-icon-button-[large,medium,small] </strong> css class name to handle styles.
  </li>
  <li>
    It should only be applied to <strong>&lt;button&gt;</strong> elements with disableRipple directive. To avoid breaking the predefined styles.
  </li>
  <li>
    White label styling is only supported <strong>without</strong> setting the "color" attribute for the <a target="_blank" href='https://v15.material.angular.io/components/button/overview'>Angular Material</a> component.
  </li>
</ol>

<PrerequisitesDoc></PrerequisitesDoc>

<p>
  Note that this directive uses icons and these are part of the @axos/uikit-v2-lib iconography. To see more information go to <a href="/docs/iconography--iconography">iconography.</a>
</p>

## Inputs

<ul>
  <InputList/>
  <li>[uk2TextButtonVariant] <u>Uk2TextButtonVariant</u> <i>(optional)</i> Control the styles for different text button variants</li>
  <li>[uk2NeutralColor] <u>boolean</u> <i>(optional)</i> Sets button color to the neutral variant</li>
</ul>

## Component-specific CSS variables

<ul>
  <li>
    <strong>[--uk2-text-icon-button-neutral-color]</strong> <i>(default: --uk2-neutral-slate-400)</i> is the neutral color of the icon for the default variant.
  </li>
  <li>
    <strong>[--uk2-text-icon-button-neutral-color-hover]</strong> <i>(default: --uk2-neutral-slate-400)</i> is the neutral color of the icon for the default variant when hovered.
  </li>
  <li>
    <strong>[--uk2-text-icon-button-neutral-color-text-variant]</strong> <i>(default: --uk2-neutral-slate-400)</i> is the neutral color of the icon for the text variant.
  </li>
  <li>
    <strong>[--uk2-text-icon-button-neutral-color-text-variant-hover]</strong> <i>(default: --uk2-neutral-slate-500)</i> is the neutral color of the icon for the text variant when hovered.
  </li>
</ul>

## Sample Code
### .component.html
<p>
  Note the use of <strong>uk2TextIconButton</strong> directive, applied to the <strong>&lt;button&gt;</strong> element.
</p>

<Source
  dark
  language='html'
  code={dedent`
    <button
      uk2TextIconButton
      mat-button
      disableRipple
      [uk2ButtonSize]="uk2ButtonSize"
      [uk2IsLoading]="uk2IsLoading"
      [disabled]="isDisabled"
    >
      <mat-icon [svgIcon]="svgIconName"></mat-icon>
    </button>
  `}
/>

### .component.ts
<p>
