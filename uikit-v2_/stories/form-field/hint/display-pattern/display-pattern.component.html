Helper Text: Display Pattern

This page showcases the following:

  • The different display patterns that can be applied to the helper text of inputs. In this case, to a text input.
  • For inputs where a helper text is specified, it can be displayed in two patterns across the range of states.
    • Not visible in default state but the present in Focus, Filled, and Disabled States. (Use hideInDefaultState option from the controls section below)
    • Visible in Default, Focus, Filled, and Disabled States. (Use alwaysVisible option from the controls section below)
  • Note that error text is different than helper text. Error text should always be present when input is in error state. And the text can/can not be the same as the helper text.
