# uk2Anchor directive documentation

## Introduction
<p> Uk2AnchorDirective is a directive that will only work on anchor elements &lt;a&gt; and it will add a class name to handle styles. </p>
<p>You just need to use uk2Anchor directive on anchor elements &lt;a&gt;.</p>
