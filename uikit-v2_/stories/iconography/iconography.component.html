Axos Iconography

UI KIT includes all icons from Axos Iconography in SVG format. That means that, once you install @axos/uikit-v2-lib you will have access to all three tiers of icons, presented in this page.

Tier 1

  • Tier 1 Icons are most commonly used at 24x24 in screens/components. They can be used at 16x16 or 20x20, 32x32 etc. Dimensions must be scaled up in increments of 8px after reaching 24x24.
  • By default, Tier 1 Icons use the color of the surrounding text. In this case set to Neutral Slate / 400. This can be easily customized by setting the color style property to the container of the icon.
  • The color of these icons can be customized with the control "Tier 1 icons color" below.

Tier 2

  • Tier 2 Icons are most commonly used at 32x23 in screens/components. They can be used at 24x24 or 32x32 etc. Dimensions must be scaled up in increments of 8px after reaching 24x24.
  • By default, Tier 2 Mono Icons use the color of the surrounding text. In this case set to Neutral Slate / 400. This can be easily customized by setting the color style property to the container of the icon. Use the control below to see how that works.
  • Tier 2 Colored with Accent are duo colored with Neutral Slate / 400 and Primary / Blue 200 accent. Neutral color can be overwritten whereas the accent cannot.

Mono

  • The color of these icons can be customized with the control "Tier 2 icons color (Mono)" below.

Colored with Accent

  • Colored with Accent Colored neutral/slate 400 or duo colored with a primary/blue 200 accent.
  • If used in a selected state, the entire icon changes color to primary/blue 400. Use the control "Tier 2 Colored with Accent is selected" below to see how it works.

Tier 3

  • Tier 3 Icons are most commonly used at 40x40. They can be used as illustrations in empty states.
  • When used in a selected state, the icon does not change color.
