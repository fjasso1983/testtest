Shimmer Animation

How this works:

Instructions

  • Use Uk2LoadingSkeleton directive or uk2-loading-skeleton__bg-[200 - 600] CSS classes to implement animation for a particular element
  • Uk2LoadingSkeleton directive will use neutral / grey colors scale for the animation. If you need your element to display a different color from the default use the CSS classes. CSS classes support neutral / grey colors from 200 to 600.
  • Note that responsiveness should be handled by the component that implements the shimmer animation.
  • Please use scss classes to hide elements within the desired element during loading state or structure control directives such as *ngIf (for this one specify what would not work)
