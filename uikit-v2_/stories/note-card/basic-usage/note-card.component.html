Basic Usage

This page showcases the following:

  • Note card states (default, skeleton)
  • Note card default sizes depending on the screen size. Change the size preview for this..
  • Customization for font size (small and large)
  • Customization for visibility of optional elements (checkbox, hyperlink)
  • Customization for hyperlink text and URL
  • Note that the hyperlink for this example it is only one and its position its fixed, but there can be many and they can be placed anywhere inside the component.
