# uk2StretchTabs directive documentation

## Introduction
<p>
  <strong>uk2StretchTabs</strong> is a custom directive developed to customize the styling of <a href="https://v15.material.angular.io/components/tabs/overview" target="_blank">&lt;mat-tab-group&gt;</a> component, to achieve Tabs component which is part of <a href="https://www.figma.com/file/xNJe50LNYYGhQ1Q1C3zs4b/%F0%9F%91%B7%E2%80%8D%E2%99%82%EF%B8%8F-Framework-Core---Web-2.0?node-id=728-10850&t=Lx4TBkfOhpDRMGVv-0" target="_blank">Axos Design System</a>.
</p>

### Constraints
<ol>
  <li>
    It should only be applied to <strong>&lt;mat-tab-group&gt;</strong> elements with <strong>mat-stretch-tabs</strong> directive, <strong>disableRipple</strong> directive and <strong>animationDuration='0ms'</strong> attribute. It will add the <strong>uk2-stretch-tabs</strong> css class name to handle styles.
  </li>
  <li>
    White label styling is only supported <strong>without</strong> setting the "color" or "defaultColor" attribute for the Angular Material component.
  </li>
</ol>

## Prerequisites

<DirectivesPrerequisitesDocsLayout></DirectivesPrerequisitesDocsLayout>

- You need to import MatTabsModule from @angular/material/* into the desired module of your Angular App. To be able to use uk2StretchTabs directive.
<Source
  dark
  language='ts'
  code={dedent`
      import { MatTabsModule } from '@angular/material/tabs';
  `}
/>

## Enums

- You need to import <strong>Uk2StretchTabsSizeEnum</strong> from @axos/uikit-v2-lib into the desired component of your Angular App. To be able to choose the size of the tab header.

<CommonEnums/>

## Sample code
  <p>
  Here are some examples on how to open uk2StretchTabs from any component.
  </p>

#### *.component.html
<Source
  dark
  code={dedent`
    <mat-tab-group
      uk2StretchTabs
      disableRipple
      [uk2StretchTabsSize]="sizeTabs"
      mat-stretch-tabs
      [uk2IsLoading]="isLoading"
      animationDuration="0ms"
    >
      <mat-tab>
        <ng-template mat-tab-label>
          <span class="uk2-stretch-tab-label">First with a long label, very very long label</span>
          <span class="uk2-stretch-tab-element-counter">(1)</span>
        </ng-template>
        <div class="fake-content-element">Fake content element first tab 1</div>
      </mat-tab>
      <mat-tab label="Second with no counter">
        <div class="fake-content-element">Fake content element second tab 1</div>
      </mat-tab>
      <mat-tab uk2Tab label="Third with counter">
        <ng-template mat-tab-label>
          <span class="uk2-stretch-tab-label">Third tabs with counter</span>
          <span class="uk2-stretch-tab-element-counter">(1)</span>
        </ng-template>
        <div class="fake-content-element">Fake content element third tab 1</div>
      </mat-tab>
    </mat-tab-group>
  `}
/>
