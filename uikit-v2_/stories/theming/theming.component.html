Theming

How this works:

  • Use the color selector in the "Controls" tab to define the primary/accent 400.
  • An HSL formula will be applied to generate the rest of monochromatic palette: primary/accent 25, 50, 100, 200, 300, 500, and 600.
  • Once the monochromatic palette is generated, all components in this website will follow the selected color scheme, following their own specifications, defined in the Axos Design System.
  • Note that the default branding for all components is Axos
  • Note the default monochromatic palette is based on Primary/blue 400 hsl(210, 49%, 36%) set as primary/accent 400.
  • Note that the palette generation will only work with hsl format. If the format of the source color is different (hex, rgb, etc) it will have to be converted into hsl. See the Docs tab for more.
