Basic Usage

This page showcases the following:

  • Tooltip component style mode (Mobile and Desktop).
  • Tooltip component customization (Label and Icon).
  • Tooltip box customization (Body and close button).
  • Tooltip box preferred placement and arrow offset.
  • Tooltip box opening mode on desktop and tablet mode.
  • Loading state of Tooltip (Skeleton mode).
  • Correct placement of the pointer can be achieved by using the "Placement Box" and "Arrow offset position" controls.
  • Custom coloring for the label and icon can be applied from the Theming menu.
