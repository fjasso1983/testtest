## Introduction
<p>
<strong>uk2ContextualTopBarDirective</strong> is a directive designed to customize the styling of <i>&lt;mat-toolbar&gt;</i> <a target="_blank" href='https://v15.material.angular.io/components/toolbar/overview'>Angular Material component</a> to adhere to <a target="_blank" href="https://www.figma.com/file/xNJe50LNYYGhQ1Q1C3zs4b/%F0%9F%91%B7%E2%80%8D%E2%99%82%EF%B8%8F-Framework-Core---Web-2.0?node-id=1000-35338&t=AlhCPNPd3UAYHVc0-0">Axos Design System</a>.
</p>

### Constraints
<ol>
  <li>
    It should only be applied to <strong>&lt;mat-toolbar&gt;</strong> Angular Material component. It will add the <strong> uk2-contextual-top-bar </strong> css class name to handle styles.
  </li>
</ol>

## Prerequisites

- You need to import MatToolbarModule from @angular/material into the desired module of your Angular App.
<Source
  dark
  language='ts'
  code={dedent`
    import { MatToolbarModule } from '@angular/material/toolbar';
  `}
/>

<DirectivesPrerequisitesDocsLayout></DirectivesPrerequisitesDocsLayout>

- You need to import Uk2Tier1NavigationEnum from @axos/uikit-v2-lib into the desired component to set the icon for the back button displayed on mobile-size screens.
  <Source
    dark
    language='ts'
    code={dedent`
      import { Uk2Tier1NavigationEnum } from '@axos/uikit-v2-lib';  `}
  />

- You need to import Uk2BreadcrumbsItem from @axos/uikit-v2-lib into the desired component.
<Source
  dark
  language='ts'
  code={dedent`
    import { Uk2BreadcrumbsItem } from '@axos/uikit-v2-lib';  `}
/>
