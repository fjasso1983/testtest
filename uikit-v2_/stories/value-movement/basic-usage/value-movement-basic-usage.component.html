Basic Usage

This page showcases the following:

  • Value movement component with customizations for type, which handles the icon to use and the color for the value part.
  • Customizations for display type.
  • Customizations for amount, percentage and description.
  • Customizations for size, it can be extraSmall, small, medium and large
  • Description. It can be plain text or projected content
  • Skeleton state. Activated when using the Is Loading control below.
