# Value movement component documentation
<br/>

## Introduction

<p> Uk2ValueMovement is a component designed on the <a href="https://www.figma.com/file/xNJe50LNYYGhQ1Q1C3zs4b/👷‍♂️-Framework-Core---Web-2.0?node-id=647%3A9378&t=pvcNxC35RkZ05y97-0" target="_blank">Axos Design System</a>. It is a reusable organism of components to provide specific context to the user related to amounts.</p>

## Prerequisites

  - You need to import Uk2ValueMovementModule from @axos/uikit-v2-lib into the desired module of your Angular App. To be able to access ValueMovementComponent.
<Source
  dark
  language='javascript'
  code={dedent`
      import { Uk2ValueMovementModule } from '@axos/uikit-v2-lib';
  `}
/>

## Enums.

  - You need to import Uk2ValueMovementTypeEnum from @axos/uikit-v2-lib to indicate the @valueMovementType property. 
<Source
  dark
  language='ts'
  code={dedent`
    import { Uk2ValueMovementTypeEnum } from '@axos/uikit-v2-lib';
    ...
    export enum Uk2ValueMovementTypeEnum {
      increasePositive = 'increase-positive',
      increaseNegative = 'increase-negative',
      decreasePositive = 'decrease-positive',
      decreaseNegative = 'decrease-negative',
      none = 'none',
      undefined = 'undefined',
    }
  `}
/>
  - You need to import Uk2ValueMovementDisplayTypeEnum from @axos/uikit-v2-lib to indicate the @displayType property. 
<Source
  dark
  language='ts'
  code={dedent`
    import { Uk2ValueMovementDisplayTypeEnum } from '@axos/uikit-v2-lib';
    ...
    export enum Uk2ValueMovementDisplayTypeEnum {
      noContainer = 'no-container',
      chip = 'chip',
      display = 'display',
    }
  `}
/>
  - You need to import Uk2ValueMovementTrendFormatEnum from @axos/uikit-v2-lib to indicate the @trendFormat property. 
<Source
  dark
  language='ts'
  code={dedent`

      triangle = 'triangle',

    }
  `}
/>
  - You need to import Uk2ValueMovementSizeEnum from @axos/uikit-v2-lib to indicate the @size property. 
<Source
  dark
  language='ts'

  `}
/>

## Description (deprecated) and projected content - Sample Code
  We still can provide [description] input to display custom text next to percentage value, but now we can show projected instead.
  Notice that [description] has a priority over projected content to keep backwards compatibility, if you provide both, [description] value will be displayed and projected content is hidden. 
  <br/>
  *.component.html

  <Source
    dark
    language='html'
    code={dedent`
      <uk2-value-movement
        [uk2IsLoading]="[boolean]"
        [description]="[string]"
        [valueMovementType]="[Uk2ValueMovementTypeEnum]"
        [isEmpty]="[boolean]"
        [isIconOnly]="[boolean]">
        <span>Projected <i>content</i></span>
      </uk2-value-movement>
    `}
  />
